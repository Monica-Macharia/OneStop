## Getting Started

Follow the steps below to set up the project on your local machine.

### Prerequisites

- PHP 7.4 or higher
- Composer
- Node.js
- NPM

### Setup

1. Fork and Clone the repository:
    cd project
    php artisan serve
    npm run dev
